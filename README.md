# 📝 Features

- Aimbot
   * Aim hotkey
   * Field-of-view
   * Aim smoothing
   * Target selector
   * Lock target
   * Switch target delay after kill
   * Draw crosshair
   * Show snaplines
   * Visibility checks
   * Aim at specific bone
   * Skip invisible
   * Flick shot
   * Prediction

- Character ESP
   * Battlemode
   * Max render distance
   * Max healthbar render distance
   * Max name render distance
   * Max info render distance
   * Healthbar position
   * Healthbar style
   * Border style
   * Max bones render distance
   * Bone style

- Radar
   * Easily spot enemies surrounding you with the 2D Radar.
   * Highly configurable!
   * Show enemies
   * Show friendly
   * Color
